## 💡 About

**RepoLoader** is a web application built for developers who want to 
quickly explore any GitHub user's repositories without manually 
visiting GitHub. Just enter a username and get everything instantly!

---

## ✨ Features

| Feature | Description |
|--------|-------------|
| 🔍 Search | Enter any GitHub username |
| 📋 Repository Table | See all repos with details |
| 💻 Language | Main language of each repo |
| 🔀 Fork Status | Know if repo is forked |
| 📅 Last Updated | Date of last update |
| 📊 Pie Chart | Visual language breakdown |
| ⬇️ CSV Download | Download data for further use |
| 📱 Responsive | Works on all devices |

---

## 🛠️ Tech Stack

- **Backend** - Python, Django
- **Frontend** - HTML, CSS, Bootstrap 5
- **Charts** - Chart.js
- **Data** - GitHub REST API
- **Export** - Pandas CSV
