# ⚡ RepoLoader Django

A web app to load all GitHub repositories of any user instantly!

## 🌐 Live Demo
Coming soon...

## 🚀 Features
- 🔍 Search any GitHub username
- 📋 See all repositories in a table
- 💻 Main language used in each repo
- 🔀 Shows forked repositories
- ⬇️ Download data as CSV
- 📊 Pie chart for languages used
- 📱 Responsive website

## 🛠️ Tech Stack
- Python
- Django
- Bootstrap 5
- Chart.js
- GitHub API
- Pandas

## 📁 Project Structure
- repoloader - Main project folder
- repoapp - App folder with views and templates
- templates - HTML files
- requirements.txt - All dependencies

## ⚙️ Installation
1. Clone the repository
2. Create virtual environment
3. Install requirements
4. Run manage.py runserver
